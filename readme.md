tenjin 是基于 FreeSWITCH 的开源呼叫中心系统，管理系统主要使用PHP开发，核心控制模块使用C语言编写。

### 主要功能和特性
* 座席队列监控
* 3种外呼模式
* 简单的订单系统
* 分机注册及状态监控
* 商品管理和语音管理
* 通话录音查询
* 通话记录和通话数据报表
* 集成VOS账户余额查询
* 可定制简单的呼入队列

### 3种外呼模式
1. 群呼转座席自动模式
2. 群呼转座席固定模式
3. 半自动一对一外呼
